# StrageFormatCLICommand

ストレージを安全かつ再現性のある手順で初期化するための CLI コマンド群。

## 🎯 目的

このディレクトリは、ストレージデバイスのフォーマット作業を  
**「整理された手順として記録し、誰でも再現できる形に残す」**  
ことを目的としています。

Github 初学者として、学んだことを整理しながら記録し、  
世界中の方と共有できるコードを少しずつ公開していきたいと考えています。

## 📂 構成

このディレクトリには以下のようなファイルが含まれます：

- フォーマット実行用 PowerShell スクリプト
- ログ生成・保存用の補助スクリプト
- 実行例・テンプレート
- 今後追加予定の CLI コマンド群

## 🛡️ 安全性

ストレージ操作は破壊的な処理を含むため、以下の方針で設計しています：

- 明示的なデバイス指定が必須
- 実行前に確認プロンプト
- ログの自動保存
- 誤操作防止のためのガード条件
